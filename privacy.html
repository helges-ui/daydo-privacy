Privacy Policy for DayDo

Last Updated: January 5, 2026

1. Information We Collect

  • Account Data: Parent and child user authentication managed via JWT tokens.
  • Family Data: Structure of family units and member profiles.
  • Features: Tasks (including recurrence), meal plans, and shared shopping lists with master item tracking.
  • Communication: Real-time family chat and messaging via WebSockets.
  • Location: Real-time location sharing and geofencing data.

2. Children's Privacy

DayDo supports PARENT and CHILD_USER roles. Child profiles are created by parents, and access is managed via a fine-grained permission system.

3. Third-Party Services

We utilize the following services to provide our app functionality:

  • AWS S3: For secure storage of recipe images and meal plan templates.
  • Mapbox: For location sharing and map visualization.
  • Infrastructure: Backend powered by Django REST API with PostgreSQL and Redis.
